General Information

Ministry User

As a Ministry User you will be able to do the following actions:

  • Create a project
  • Edit and save the project
  • Submit the project for SID feedback
  • Check the status of the project

SID User

As a SID User you will be in charge of:

  • Checking the status of submitted projects
  • Deem the project idea comprehensive/satisfactory or unsatisfactory
  • Download pdf reports

Disclaimer

The submission of project ideas or information provided by SID as part of this web application does not constitute in any way, approval in terms of EU funding, and does not pre-judge the decisions or feedback provided by designated funding evaluation entities, should the project proposal be submitted for funding. In this regard, any feedback provided by the SID team should be considered as an advisory feedback with the aim of improving capacity in terms of drafting project proposals. Also, it should be noticed that proposals submitted to SID through the web application will need to be further revisited at EU funding application submission stage taking into consideration the applicable requirements of the call and also taking into consideration the time span between the ‘accepted proposal’ and the submitted application since needs, context etc. may change. Furthermore, calls would have additional requirements strictly relating to the OPs/funding policy. Therefore, it should not be assumed that the information provided through this web application would be enough to fill in an application for funding.
